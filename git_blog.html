Git && Git Commands

Let's learn about git pull and push.

Git?

Git is a version control system. It helps you keep track of code changes. It was created by Linus Torvalds(2005) and maintained by  Junio Hamano.


Purpose of Git

  • Manage projects with different Repositories.
  • Clone a project to work on a local copy.
  • Control and track changes with different environment  and Committing.
  • Create Branch and Merge the updates to your repositories.
  • Pull the latest version of the project to a local copy.
  • Push local updates to the project.


Installation & Commands

Install the git in your local machine as instructed in Official website of git. You can pick it from here.

Let we move to commands.

1. Git Clone

Git Clone

  • Clone your repo in your machine.

    • command:
     ​  git clone -b 'repo link(https)'

    2.Git Branch

    Git Branch

  • Create a new branch for your repo.

    • Eg: staging, develop

    command:
     ​  ​git branch 'branch name'

    • To list all branch : git branch -a
    • To move a specific branch : git branch 'branch name'
    • To delete the branch : git branch -d 'branch name'
    • To force delete the branch : git branch -D 'branch name'
    • To rename branch : git branch -m 'branch name'

    3. Git Checkout

    Git Checkout

  • By using this we can create branch and switch to the branch immediately.

    • command:
     ​  git checkout -b 'branch name'


  • To switch to another branch

    • command:
     ​  git checkout 'branch name'

    4. Git Pull

    Git Pull

  • Git pull is used to get or pull the latest changes from the repo to your local.

    • command:
     ​  git pull origin 'branch name'

     ​  or
     ​  git pull

    Note: After checkout to the new branch, used git pull command to get the latest changes from respected branches.

    5. Git Add

    Git Add

  • Git add is used to add the modified file or new file to the repo. Below command used to all files. in the repo.

    • command:
     ​  git add --all

    • Alternative for adding all files: git add -A
    • Add the entire directory and also the file starts with .(dot) : git add .
    • Add the modified and deleted file, not includes new file : git add -u
    • To add specific file: git add 'file path'

    5. Git Status

    Git Status

  • Git status is used to check the file that have been added, we can verify the file that we have changed.

    • command:
     ​  git status

    6. Git Commit

    Git Commit

  • Git commit is used to commit the file to the repo.

    • command:
     ​  git commit -m 'Description'

    7. Git Push

    Git Push

  • This command is used to push your changes to your repo.

    • command:
     ​  git push origin 'branch name'

    Overall Steps

    • git clone -b 'repo link'
    • git checkout -b ''branch name'
    • git pull origin 'branch name'
    • git add --all
    • git status
    • git commot -m 'description'
    • git push origin 'branch name'
